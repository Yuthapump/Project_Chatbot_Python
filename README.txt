# Line Chatbot Kamkin Muang 

                         เรียนทุกท่านที่สนใจใช้งาน Line Chatbot อาหารเหนือของเรา
                         จุดประสงค์เพื่อให้ทุกท่านได้รับประสบการณ์ที่เต็มเปี่ยมด้วยความสะดวกสบาย
                         และมีประโยชน์ในการสำรวจและสัมผัสกับอาหารเหนืออย่างใกล้ชิด    
                         รวมไปถึงการรวบรวมอาหารเหนือขึ้นชื่อจำนวนนึงเพื่อให้ผู้ใช้สามารถที่จะเลือก
                         และตัดสินใจได้อย่างสะดวกและสบายใจที่สุดจ้าว :)

# เป็นการประยุกต์ใช้ Flex Message ในการเขียน code

[4] ทำการ กดที่ค้าหา
